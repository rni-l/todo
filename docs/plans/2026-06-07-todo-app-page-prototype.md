# 个人 TODO 应用页面原型文档

日期：2026-06-07
用途：用于生成页面原型、信息架构图、高保真设计稿和后续实现计划
产品方向：个人自托管 TODO 网页应用
设计方向：Apple-ish 个人效率工作台
参考文档：`docs/plans/2026-06-07-todo-app-prototype-spec.md`

## 1. 文档目标

本文档在上一版产品原型规格基础上，继续细化到页面级。重点回答三个问题：

1. 应用第一版需要哪些页面和弹层。
2. 每个页面包含什么内容、按钮、状态和功能逻辑。
3. 页面之间如何跳转、打开、返回、保存和联动。

本文档面向设计稿生成工具和产品原型设计，不优先描述代码实现。所有页面均按单用户自用、云服务器部署、在线优先、PWA 可安装的前提设计。

## 2. 设计规范基线

### 2.1 产品界面判断

这是一个长期自用的个人生产力工具，不是营销页、展示站或团队 SaaS 控制台。视觉上应该像一个安静、顺手、有质感的个人工作台。

关键词：

- 私密
- 安静
- 清晰
- 高效
- 柔和材质
- 轻量玻璃感
- 状态明确
- 可长期高频使用

避免：

- 大面积营销页 Hero
- 夸张滚动动画
- 装饰性 bento 卡片
- 紫蓝 AI 渐变
- 过度玻璃拟态
- 过度可爱空状态
- 企业后台式密集表格

### 2.2 设计密度

桌面端信息密度偏高，但不能压迫。

- 桌面任务行高度：44 到 56 px
- 移动任务行高度：56 到 72 px
- 主要点击目标：不低于 40 px
- 移动端触控目标：不低于 44 px

### 2.3 页面结构原则

所有登录后的页面共用同一个工作台外壳：

- 桌面：左侧导航 + 中间内容 + 右侧任务详情抽屉
- 移动：顶部标题栏 + 主内容 + 底部导航 + 全屏任务详情页

页面之间尽量不打断当前上下文。任务详情、日期选择、优先级选择、附件操作、命令面板等优先使用抽屉、弹层或底部 sheet。

## 4. 全局工作台外壳

### 4.1 桌面端工作台

适用页面：

- 所有登录后的一级页面

布局：

- 左侧导航栏固定宽度 248 px
- 中间内容区自适应
- 右侧任务详情抽屉宽度 420 到 520 px

左侧导航内容：

- 应用标识
- 新建任务按钮
- 主要导航：今日、收件箱、即将到来、日历
- 组织导航：项目、标签、智能过滤器
- 历史导航：已完成
- 底部入口：设置、主题切换、账号菜单

左侧按钮：

- `新建任务`
- `展开/收起项目组`
- `展开/收起智能过滤器组`
- `设置`
- `退出登录`

中间内容区通用结构：

- 页面标题
- 页面副标题或任务数量
- 快速添加输入框，部分页面显示
- 视图控制，如排序、分组、筛选、周/月切换
- 主内容列表或日历

右侧任务详情抽屉逻辑：

- 点击任务行打开
- 再点击其他任务时直接切换内容
- 按 Escape 关闭
- 有未保存输入时显示保存中或已保存状态
- 删除任务后关闭抽屉并刷新列表

## 5. 页面详细说明

## 5.1 登录页

页面目标：

- 验证单用户身份
- 阻止未登录访问任务数据

入口：

- 未登录访问任意登录后页面
- 会话过期后自动跳转

页面内容：

- 应用名称
- 简短说明文案
- 密码输入框
- 可选用户名输入框
- 登录按钮
- 错误提示区域
- 服务器连接状态提示

主要按钮：

- `登录`

辅助按钮：

- `显示/隐藏密码`
- `重试连接`，仅服务器不可用时显示

功能逻辑：

- 用户输入密码后点击登录或按 Enter
- 登录中按钮进入 loading 状态
- 登录成功后进入今日页
- 登录失败显示错误，不清空已输入的用户名
- 服务器不可用时显示连接错误
- 已登录用户访问登录页时自动跳转今日页

状态：

- 默认
- 输入聚焦
- 登录中
- 密码错误
- 服务器不可用
- 会话已存在

跳转：

- 登录成功 -> 今日页
- 服务器不可用 -> 留在登录页

## 5.2 今日页

页面目标：

- 展示当天要处理的所有任务
- 支持快速完成、延期、改优先级和打开详情

入口：

- 登录成功后的默认落点
- 侧边栏 `今日`
- 移动端底部导航 `今日`
- 命令面板跳转

页面内容：

- 页面标题：今日
- 日期，如 2026年6月7日 周日
- 未完成任务数量
- 快速添加输入框
- 任务分组：
  - 已逾期
  - 今天
  - 无提醒时间
  - 今日已完成，默认折叠

顶部按钮：

- `新建任务`
- `搜索`
- `排序`
- `分组`
- `更多`

任务行按钮：

- 完成复选框
- 打开详情
- 改日期
- 改优先级
- 更多操作

更多操作：

- 明天再做
- 移到项目
- 添加标签
- 删除

功能逻辑：

- 在今日页快速添加任务时，默认截止日期为今天
- 点击完成后，任务出现短暂完成反馈，然后移动到今日已完成
- 逾期任务可一键延期到今天、明天或自定义日期
- 拖拽任务可调整今日页内手动排序
- 点击任务标题或行主体打开任务详情抽屉
- 点击日期 chip 打开日期选择弹层
- 点击优先级标记打开优先级选择弹层

空状态：

- 文案：今天没有待办
- 辅助文案：可以休息，也可以添加一个今天要处理的任务
- 主按钮：添加任务

页面跳转：

- 点击任务 -> 任务详情抽屉
- 点击项目 chip -> 项目详情页
- 点击标签 chip -> 标签详情页
- 点击日期 chip -> 日期选择弹层
- 点击搜索 -> 搜索/命令面板

## 5.3 收件箱页

页面目标：

- 接收尚未整理到项目的任务
- 作为默认快速捕获容器

入口：

- 侧边栏 `收件箱`
- 移动端底部导航 `收件箱`
- 命令面板跳转
- 新建任务未指定项目时默认进入收件箱

页面内容：

- 页面标题：收件箱
- 未完成数量
- 快速添加输入框
- 未整理任务列表
- 已完成任务折叠区，可选

顶部按钮：

- `新建任务`
- `整理`
- `排序`
- `更多`

整理按钮逻辑：

- 进入批量整理模式
- 可多选任务
- 批量移动到项目
- 批量添加标签
- 批量设置日期

任务行功能：

- 完成
- 打开详情
- 设定日期
- 移动到项目
- 添加标签
- 删除

空状态：

- 文案：收件箱已清空
- 辅助文案：新的想法和待办可以先放在这里
- 主按钮：添加任务

页面跳转：

- 任务 -> 任务详情抽屉
- 批量移动 -> 项目选择弹层
- 添加标签 -> 标签选择弹层

## 5.4 即将到来页

页面目标：

- 按日期分组查看未来任务
- 支持轻量计划与改期

入口：

- 侧边栏 `即将到来`
- 命令面板跳转

页面内容：

- 页面标题：即将到来
- 未来任务总数
- 日期分组：
  - 明天
  - 本周
  - 下周
  - 以后
  - 无日期，可选

顶部按钮：

- `新建任务`
- `显示无日期`
- `排序`
- `更多`

任务操作：

- 拖拽到其他日期分组
- 点击日期 chip 改日期
- 点击任务打开详情
- 完成任务

功能逻辑：

- 任务拖到某日期分组后更新截止日期
- 分组可以折叠
- 无日期分组默认隐藏，可通过按钮显示
- 已完成任务默认不显示

页面跳转：

- 任务 -> 任务详情抽屉
- 日期分组标题 -> 日历页对应日期
- 项目 chip -> 项目详情页

## 5.5 日历页

页面目标：

- 用周/月视图查看任务日期分布
- 通过拖拽调整任务截止日期

入口：

- 侧边栏 `日历`
- 移动端底部导航 `日历`
- 命令面板跳转
- 从今日/即将到来的日期入口跳转

页面内容：

- 页面标题：日历
- 当前周或当前月份
- 周/月切换控件
- 上一段/下一段按钮
- 回到今天按钮
- 日历网格

顶部按钮：

- `今天`
- `上一周/上一月`
- `下一周/下一月`
- `周`
- `月`
- `新建任务`
- `更多`

周视图内容：

- 7 个日期列
- 每列显示星期、日期、任务数量
- 当天高亮
- 任务以紧凑行或 pill 显示

月视图内容：

- 月历网格
- 每个日期格显示日期号、任务数量、前几条任务
- 超出显示 `还有 N 项`
- 点击日期格进入当天任务列表或日详情弹层

任务操作：

- 拖拽任务到其他日期
- 点击任务打开详情
- 点击空白日期新建该日期任务
- 点击 `还有 N 项` 查看当天全部任务

功能逻辑：

- 日历只按截止日期展示任务
- 提醒时间作为小文字显示，不形成时间块
- 任务没有开始时间和结束时间
- 拖拽改期后立即刷新今日、即将到来和任务详情中的日期
- 重复任务显示重复标记

页面跳转：

- 点击日期 -> 日期任务弹层或即将到来筛选结果
- 点击任务 -> 任务详情抽屉
- 周/月切换 -> 当前日历页面内状态切换

## 5.6 项目总览页

页面目标：

- 查看所有项目
- 创建、排序、归档和管理项目

入口：

- 侧边栏 `项目`
- 移动端底部导航 `项目`
- 更多页中的项目入口

页面内容：

- 页面标题：项目
- 项目列表
- 每个项目显示：
  - 项目名称
  - 未完成任务数量
  - 今天到期数量
  - 最近更新时间
  - 可选项目颜色或图标

顶部按钮：

- `新建项目`
- `排序`
- `显示已归档`

项目行按钮：

- 打开项目
- 重命名
- 归档
- 删除
- 拖拽排序

功能逻辑：

- 点击项目进入项目详情页
- 新建项目打开项目创建弹层
- 删除项目需要确认
- 删除项目时需要选择：
  - 同时删除任务
  - 将任务移回收件箱
- 归档项目后不在默认列表显示

空状态：

- 文案：还没有项目
- 主按钮：新建项目

页面跳转：

- 项目行 -> 项目详情页
- 新建项目 -> 项目创建弹层

## 5.7 项目详情页

页面目标：

- 管理某个项目中的任务和章节

入口：

- 项目总览页点击项目
- 侧边栏项目快捷项
- 任务详情中的项目字段
- 命令面板

页面内容：

- 项目标题
- 项目说明或任务数量
- 快速添加输入框
- 章节列表
- 每个章节下的任务列表

顶部按钮：

- `新建任务`
- `新建章节`
- `排序`
- `分组`
- `项目设置`

项目设置菜单：

- 重命名项目
- 更改颜色
- 归档项目
- 删除项目

章节按钮：

- 重命名章节
- 新建任务到章节
- 折叠章节
- 删除章节
- 拖拽排序章节

任务逻辑：

- 在项目详情页快速添加任务时，默认项目为当前项目
- 在某章节中添加任务时，默认章节为当前章节
- 任务可拖拽到其他章节
- 章节删除时，可选择将任务移动到无章节或一起删除

空状态：

- 项目空：这个项目还没有任务
- 章节空：这个章节是空的

页面跳转：

- 任务 -> 任务详情抽屉
- 标签 -> 标签详情页
- 日期 -> 日期选择弹层

## 7. 页面之间的逻辑关系

### 7.1 登录与鉴权逻辑

1. 未登录访问任何工作台页面时，跳转登录页。
2. 登录成功后，默认进入今日页。
3. 登录状态有效时，访问登录页自动跳转今日页。
4. 会话过期时，当前页面显示会话过期弹层。
5. 用户确认后回到登录页。
6. 重新登录后默认回到之前访问的页面，若无法恢复则进入今日页。

### 7.2 主导航逻辑

桌面侧边栏是主导航源。

- 今日 -> 今日页
- 收件箱 -> 收件箱页
- 即将到来 -> 即将到来页
- 日历 -> 日历页
- 项目 -> 项目总览页
- 项目快捷项 -> 项目详情页
- 标签 -> 标签总览页
- 智能过滤器 -> 智能过滤器总览页
- 智能过滤器快捷项 -> 智能过滤器详情页
- 已完成 -> 已完成页
- 设置 -> 设置首页

移动端底部导航只放最高频入口。

- 今日 -> 今日页
- 收件箱 -> 收件箱页
- 日历 -> 日历页
- 项目 -> 项目总览页
- 更多 -> 移动端更多页

更多页承接低频入口：

- 即将到来
- 标签
- 智能过滤器
- 已完成
- 设置

### 7.3 任务详情打开逻辑

桌面：

1. 用户点击任意任务行。
2. 右侧任务详情抽屉打开。
3. 用户继续点击其他任务，抽屉内容切换。
4. 用户关闭抽屉后回到当前列表。

移动：

1. 用户点击任务行。
2. 进入任务详情全屏页。
3. 点击返回回到来源页。
4. 若任务修改后不再匹配来源列表，返回时列表刷新。

### 7.4 新建任务逻辑

上下文决定默认值：

- 今日页：默认截止日期今天
- 收件箱页：默认无项目
- 项目详情页：默认当前项目和章节
- 标签详情页：默认当前标签
- 日历日期格：默认对应日期
- 智能过滤器详情页：不强行套用全部过滤条件，只套用明确的项目、标签或日期条件

流程：

1. 用户在快速添加框输入标题。
2. 按 Enter 创建。
3. 任务出现在当前上下文中。
4. 输入框清空并保持聚焦。
5. 用户可立即继续输入下一条。

### 7.5 日期与日历联动逻辑

1. 在任何页面修改任务截止日期后，今日页、即将到来页、日历页和智能过滤器都应同步反映。
2. 日历拖拽任务到其他日期，本质是修改任务截止日期。
3. 任务没有截止日期时，不显示在日历中。
4. 任务有提醒时间但无截止日期时，需要引导设置截止日期，或将提醒日期视为任务日期，后续实现确定。

### 7.6 项目与章节联动逻辑

1. 任务只能属于一个项目。
2. 任务可以属于该项目下的一个章节。
3. 从项目 A 移动到项目 B 时，原章节清空，除非选择目标章节。
4. 删除章节时，任务可移动到无章节。
5. 删除项目时，用户必须选择处理项目内任务的方式。

### 7.7 标签联动逻辑

1. 任务可以拥有多个标签。
2. 标签删除后，只移除任务上的该标签，不删除任务。
3. 标签详情页中移除任务的当前标签后，该任务从当前页面消失。
4. 智能过滤器中引用已删除标签时，应显示条件失效提示。

### 7.8 智能过滤器联动逻辑

1. 智能过滤器不是任务容器，只是保存的视图。
2. 任务属性变化后，过滤器结果实时变化。
3. 在过滤器页面创建任务时，仅自动应用明确且安全的上下文条件。
4. 如果过滤器包含复杂条件，例如有附件、已逾期，不自动套用到新任务。
5. 删除过滤器不影响任务。

### 7.9 附件联动逻辑

1. 附件只能挂在任务上。
2. 附件上传成功后，任务行显示附件标记。
3. 删除附件只删除附件，不删除任务。
4. 删除任务时，需要同时删除任务附件或将附件变成不可见资源，第一版建议随任务删除。
5. 数据导入后如果附件文件未导入，任务详情显示文件未恢复。

### 7.10 设置与全局状态联动逻辑

1. 外观设置变化后，全应用即时生效。
2. 通知权限变化后，提醒相关 UI 更新。
3. 数据导入成功后，全局任务数据刷新。
4. 退出登录后清空当前前端会话状态，回到登录页。

## 8. 核心用户路径

### 8.1 每天打开应用

1. 用户打开 PWA 或网页。
2. 如果未登录，进入登录页。
3. 登录后进入今日页。
4. 用户处理逾期和今日任务。
5. 用户打开详情补充信息或完成任务。

### 8.2 快速记录一个任务

1. 用户在任意主页面看到快速添加框。
2. 输入任务标题。
3. 按 Enter。
4. 任务创建并出现在当前上下文。
5. 用户继续输入下一条或打开详情补充日期、标签、附件。

### 8.3 从日历规划任务

1. 用户进入日历页。
2. 切换周视图或月视图。
3. 拖拽任务到新日期。
4. 任务日期更新。
5. 今日页和即将到来页同步变化。

### 8.4 创建一个项目并整理任务

1. 用户进入项目总览页。
2. 点击新建项目。
3. 输入项目名称并保存。
4. 进入项目详情页。
5. 新建章节。
6. 从收件箱移动任务到项目或直接在项目中创建任务。

### 8.5 创建智能过滤器

1. 用户进入智能过滤器总览页。
2. 点击新建过滤器。
3. 输入名称。
4. 添加条件。
5. 选择排序和分组。
6. 查看预览。
7. 保存。
8. 进入智能过滤器详情页。

### 8.6 给任务添加附件

1. 用户打开任务详情。
2. 点击添加附件。
3. 选择文件。
4. 应用检查文件大小。
5. 上传成功后显示附件卡片。
6. 任务行显示附件标记。

### 8.7 导出备份

1. 用户进入设置。
2. 打开数据导入导出页。
3. 导出 JSON 数据。
4. 打开附件导入导出页。
5. 导出附件 ZIP。
6. 用户自行保存两个文件。
